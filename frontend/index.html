Siphon HEX-H13

Módulo purificador de aire con planta marina y monitoreo IoT en tiempo real

  • ESP32
    Microcontrolador con WiFi
  • DHT11
    Sensor de temperatura y humedad
  • TCS230
    Sensor de color para pH
  • Planta Marina
    Purificación natural del aire

Siphon utiliza una planta marina dentro de un módulo hexagonal para purificar el aire de forma natural. Sensores IoT monitorean continuamente la temperatura, humedad y pH del agua, enviando los datos a un servidor local para visualización en tiempo real.
