¿Qué es este módulo?

Este es un módulo hexagonal purificador de aire que utiliza un alga marina (Ulva) para absorber CO₂ del ambiente mediante la fotosíntesis. El módulo monitorea la calidad del aire y la salud de la planta usando tres sensores conectados a un ESP32.

Sensores incluidos

  • DHT11 — Mide temperatura y humedad ambiente.
  • MQ2 — Detecta gases y calcula un índice de calidad de aire.
  • TCS230 — Lee el color de una tira indicadora de pH para estimar el pH del agua.

¿Cómo se usa?

  1. Colocá el módulo en una superficie metálica (los imanes lo sostienen).
  2. Conectalo a la corriente mediante el cable USB-C.
  3. Escaneá el código QR para abrir esta aplicación web.
  4. Monitoreá la temperatura, humedad, calidad del aire y pH de la planta.

Tecnología

  • ESP32 con WiFi — envía los datos a un servidor Node.js.
  • Backend en Node.js — recibe y guarda las lecturas.
  • App web responsiva — funciona en cualquier celular sin instalar nada.
